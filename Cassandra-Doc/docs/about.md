# StatefulSets

## Using StatefulSets

We need statefulSets for the following

- Unique network identifiers.
- Persistent storage.
- Deployments.
- Scaling.
- Rolling Updates.


## Cassandra Statefulsets Object Specs

1. Pod Selector
2. Ordinal Index
3. Network ID
4. Pod Identity 
5. Pod Name Label 
6. Pod Management Policies
7. OrderedReady Pod Management
8. Parallel Pod Management
9. Update Strategies
10. On Delete
11. Rolling Updates
12. Forced Rollback
